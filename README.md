# INTRODUCTION

Prompt Engineering is the process of creating effective prompts to improve the quality of AI-generated responses. Different prompt styles influence the clarity, accuracy, and depth of AI outputs. This experiment compares naïve prompts, which are broad and unstructured, with basic prompts, which are clear and refined.

The experiment evaluates AI responses in different scenarios such as:

- Creative story generation  
- Factual question answering  
- Summarization  
- Recommendations and advice  

The outputs are compared using evaluation metrics like quality, accuracy, and depth.

# DEFINING THE TWO PROMPT TYPES

## 1. Naïve Prompt

A naïve prompt is short, vague, or unstructured with minimal instructions.

### Example

```text
Tell me about Artificial Intelligence.
```

## 2. Basic Prompt

A basic prompt is clear, structured, and contains context or detailed instructions.

### Example

```text
Explain Artificial Intelligence in simple terms with examples, applications, advantages, and disadvantages.
```

# OBSERVATIONS

- Basic prompts consistently produced better results.  
- Structured prompts improved response quality and detail.  
- Naïve prompts worked for simple tasks but lacked depth.  
- Prompt clarity directly influenced AI performance.  

# ADVANTAGES OF BASIC PROMPTS

- Better response quality  
- Improved accuracy  
- More detailed explanations  
- Better contextual understanding  
- Structured outputs  

# LIMITATIONS OF NAÏVE PROMPTS

- Ambiguous responses  
- Less detail  
- Lower contextual understanding  
- Inconsistent output quality  

# CONCLUSION

The experiment demonstrated that prompt clarity significantly affects AI-generated responses. Naïve prompts generated short and less detailed outputs, while basic prompts produced accurate, comprehensive, and structured responses. Comparative analysis showed that well-structured prompts improve the quality, depth, and relevance of AI outputs across different scenarios.
